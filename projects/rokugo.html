rokugo 六五

Rokugo is the first part of my NES (or custom console) emulator - the 6502 processor! Written in Rust

I'm basing this design on the static w65c02 as that's the version I have as a real chip at home

The display is a debugger which decompiles the machine code into a human readable format

I have an idea to overclock this and use it as a graphics processor (ppu) considering old NES ppu units really aren't available.

Making sure my design works emulated will help me in the hardware step (as syncing clock cycles and r/w to shared memory will be much harder there)

Then I can make my own custom 8 bit console

One day I'd like to build my own computer out of logic gates so making an emulator is a good start in understanding everything that goes on behind the scenes
